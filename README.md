## ğŸš€ Getting Started

To get a local development copy of the Clerix Dashboard up and running, follow these steps.

**Prerequisites**
*   Node.js 18+ installed

**Installation**

1.  **Clone the repo**
    ```bash
    git clone https://github.com/manandevs/clerix.git
    cd clerix
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  Open `http://localhost:3000` in your browser.
